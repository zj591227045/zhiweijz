## 升级说明

### 升级步骤

```bash
# 1. 获取最新代码
cd zhiweijz
git pull

# 2. 备份数据库（可选但推荐）
cp scripts/database/db_backup/config.conf.template scripts/database/db_backup/config.env
# 编辑配置文件，设置数据库用户名和密码
vim scripts/database/db_backup/config.env
./scripts/database/db_backup/backup.sh full

# 3. 拉取最新docker镜像并更新
cd docker
docker-compose -p zhiweijz pull
docker-compose -p zhiweijz up -d
```

## 新功能详解

### 图片识别多记录选择导入
- 支持从单张图片识别多条记账记录
- 用户可以选择性导入识别出的记录
- 提供记录编辑和确认功能
- 支持批量操作，提升记账效率

### 安卓快捷记账
- 桌面快捷方式快速启动记账
- 简化记账流程，减少操作步骤
- 支持语音快速输入
- 智能分类推荐

### 分摊计算功能
- 支持多人共同支出记录
- 自动计算个人分摊金额
- 支持不同分摊方式（平均、按比例、自定义）
- 分摊记录清晰展示和统计

### 火山方舟大模型集成
- 更精准的记账分类识别
- 更好的语义理解能力
- 支持更复杂的记账场景
- 提供记账建议和预算提醒
