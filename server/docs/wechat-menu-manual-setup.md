# 微信公众号菜单手动配置指南

如果遇到IP白名单问题，可以通过微信公众号后台手动创建菜单。

## 方法一：微信公众号后台手动配置

### 步骤：
1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入"自定义菜单"
3. 创建以下菜单结构：

### 菜单配置：

#### 菜单1：访问官网
- **菜单名称**: 访问官网
- **菜单类型**: 跳转网页
- **页面地址**: https://www.zhiweijz.cn

#### 菜单2：账号绑定  
- **菜单名称**: 账号绑定
- **菜单类型**: 跳转网页
- **页面地址**: https://www.zhiweijz.cn

#### 菜单3：下载App
- **菜单名称**: 下载App
- **菜单类型**: 跳转网页
- **页面地址**: https://www.zhiweijz.cn/downloads

### 发布菜单：
4. 点击"发布"按钮
5. 菜单将在5分钟内生效

## 方法二：解决IP白名单问题

### 需要添加的IP地址：
```
82.152.164.115
144.255.19.94
```

### 配置步骤：
1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入"开发" → "基本配置"
3. 在"IP白名单"中添加上述IP地址
4. 保存配置
5. 等待5-10分钟生效
6. 重新运行脚本：`npm run create-wechat-menu`

## 方法三：使用微信开发者工具测试

可以使用微信官方提供的[接口调试工具](https://mp.weixin.qq.com/debug/)来测试菜单创建：

### 请求参数：
```json
{
  "button": [
    {
      "type": "view",
      "name": "访问官网",
      "url": "https://www.zhiweijz.cn"
    },
    {
      "type": "view", 
      "name": "账号绑定",
      "url": "https://www.zhiweijz.cn"
    },
    {
      "type": "view",
      "name": "下载App",
      "url": "https://www.zhiweijz.cn/downloads"
    }
  ]
}
```

## 验证菜单是否生效

1. **取消关注公众号**
2. **重新关注公众号**
3. **查看底部菜单是否显示**

或者等待5-10分钟后直接查看菜单。

## 注意事项

- 菜单更新可能需要5-10分钟生效
- 建议用户取消关注后重新关注以立即看到新菜单
- 确保URL地址可以正常访问
- 菜单名称不能超过16个字符
